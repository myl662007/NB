## git常用基本操作

- git拉代码
        
        git pull

- git推代码
        
        git push
        
- git处理冲突合并
        
        git stash save 'myl'
        git pull
        git stash pop
        git push
        
- git克隆
    
        git clone http://git.sxjicheng.com/jicheng/rtu_center.git

- 查看状态
        
        git status
