- 算术运算符
    
        注意：两个整数相除，结果会去掉小数部分
        ++ -- 只有后加加和后减减，必须单独使用 

- 赋值


- 比较运算符（关系运算符）
        

- 逻辑

        &&
        ||

- 位
        

- 其他
    
        & 取地址
        * 指针
        
        案例：
            func main(){
            	a := 100
            	fmt.Println("a的地址",&a)  //地址
            
            	var ptr *int = &a
            	fmt.Println("ptr 指向的是",*ptr) //100
            }

注意：go语言中没有三元运算符
