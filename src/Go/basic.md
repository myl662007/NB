## Basic
- go的相关知识点

        go是一个静态类型的编译型语言，安全和性能
        
        代码包相关知识：
        作用：编译和归档go程序的最基本单位
        
        fmt.Println("wode")不能用单引号
        
        源代码文件分为3种：
        源代码文件、库源码文件和测试源代码文件

- Go语言的基本命令及使用
    - go command [arguments]
        
          包含的命令如下：
          build 编译源代码包和依赖
          run 先编译源码文件再运行
          clean 删除对象文件
          doc 显示go包或程序实体的文档
          env 打印输出go语言环境的变量信息
          install 编译并安装指定的包与依赖
