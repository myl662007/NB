##TCP协议

- TCP/IP

        TCP/IP(Transmission Control Protocol/Internet Protocol)的简写,中文译名为传输控制协议/因特网互 联协议，
        又叫网络通讯协议，这个协议是 Internet 最基本的协议、Internet 国际互联网络的基础，简单 地说，
        就是由网络层的 IP 协议和传输层的 TCP 协议组成的

- Tcp socket 编程

- b/s 结构的 http 编程

- 端口

        1-1024是固定端口（程序员不要使用）
        
        1025-65535是动态端口
